# mrp

Install git

Install gradle

Install Node and npm

Update node and npm to the latest stable version :

    $ sudo npm install -g n
    $ sudo n stable

Install bower :

    $ sudo npm install -g bower
    
----

Get the source code :

    $ git clone https://github.com/ymauray/mrp.git
    $ cd mrp
    
Get the javascript components :

    $ bower install
    
Run and enjoy !
